This repo contains some examples of simple nodejs HTTP servers.

http://nodejs.org/

## Installing npm

###native node

```bash
cd wtwnodetutorial/nativenodeserver
npm install
node server
```

###connect

```bash
cd wtwnodetutorial/connectserver
npm install
node server
```

###express.js

```bash
cd wtwnodetutorial/expressserver
npm install
node server
```
